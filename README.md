## Bookjournalism Clone Project with TDD
- CRA와 RTL(React Testing Library), Jest를 통한 TDD 방식의 프로젝트
- 테스트코드가 없으면 구현하지 않는다는 TDD 원칙에 입각하여 진행
- UI 구현에 초점을 맞추는 React의 철학에 따라 DOM 위주의 테스트
